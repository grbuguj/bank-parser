# 🏦 은행 거래내역 파서

개인회생 사건용 은행 거래내역서 자동 추출 시스템

## 지원 은행
- NH뱅크
- 카카오뱅크
- 케이뱅크
- 토스뱅크
- KB와이즈
- 기타 (범용)

## 설치 방법

```bash
# 1. 패키지 설치
pip install -r requirements.txt

# 2. 실행
streamlit run app.py
```

## 사용 방법

1. 브라우저에서 `http://localhost:8501` 접속
2. 왼쪽 사이드바에 OpenAI API Key 입력
3. 은행 선택
4. PDF 업로드
5. 필터 금액 설정 (기본: 50만원)
6. 실행 버튼 클릭
7. 엑셀 다운로드

## 출력 컬럼
| 컬럼 | 설명 |
|------|------|
| 거래은행 | 은행명 |
| 입금일 | 입금 거래일 (출금이면 빈칸) |
| 출금일 | 출금 거래일 (입금이면 빈칸) |
| 금액 | 거래 금액 |
| 거래사유 | AI 추론 (노란 배경 = 추측) |

## 주의사항
- OpenAI API 비용 발생 (PDF 1개당 약 200~400원)
- 스캔 품질이 낮으면 정확도 저하될 수 있음
- 거래사유는 AI 추측이므로 반드시 검토 필요
